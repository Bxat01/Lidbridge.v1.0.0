# LidBridge Private Key
#
# This directory should contain your GitHub App private key.
# The file must be named: private-key.pem
#
# To generate or download your private key:
#   1. Go to https://github.com/settings/apps
#   2. Select your GitHub App
#   3. Scroll to "Private keys"
#   4. Click "Generate a private key"
#   5. Place the downloaded .pem file here as: private-key.pem
#
# IMPORTANT: Never commit private-key.pem to version control.
# It is already listed in .gitignore for your protection.
